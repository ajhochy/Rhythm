# Automation Rules (Initial)

- Recurring task rules generate concrete tasks ahead of active planning windows.
- Recurring annual project templates generate fresh project instances each cycle.
- Project template steps may be expressed as offsets relative to a target due date.
- Weekly planner inclusion set: one-off tasks + recurring instances + project-generated tasks + calendar shadow events.
- Automation rules should be deterministic and idempotent for repeatable regeneration.
