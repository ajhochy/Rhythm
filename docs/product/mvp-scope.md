# MVP Scope

## In scope
- One-off task capture and scheduling
- Recurring task rules (weekly/monthly/annual) with instance generation
- Recurring annual project templates and project instance generation
- Project template steps with due-date offsets
- Weekly planner view combining all generated/planned work
- Calendar shadow events ingestion (read-only in MVP)

## Out of scope
- Mobile-native app experience
- Full bidirectional external sync editing
- Team collaboration permissions model
- Advanced automation scripting UI

## Phase 1 priorities
1. Stable domain model and instance generation pipeline
2. Weekly planner UX with desktop multi-pane layout
3. Baseline integrations surface with explicit source-of-truth boundaries
4. Clear service/repository architecture for future expansion
