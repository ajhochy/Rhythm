# User Flows

1. **Capture one-off task** → assign due date/week bucket → appears in weekly planner.
2. **Create recurring task rule** → background generation creates upcoming concrete tasks.
3. **Create annual project template** → set target date + offset steps → yearly project instance created.
4. **Weekly planning session** → review one-offs + recurring instances + project tasks + calendar shadows.
5. **Review integration signals** → open linked context while preserving Rhythm as planning source.
