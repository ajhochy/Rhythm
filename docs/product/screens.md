# Screen Concepts (Desktop-first)

## Navigation sidebar
Persistent left rail for Weekly Planner, Tasks, Projects, Rhythms, Integrations, Settings.

## Weekly planner main pane
Primary center pane listing the current week’s planned workload, grouped by day and priority.

## Detail pane
Right-side pane for task/project detail, links, and scheduling adjustments without navigation loss.

## Project timeline view
Desktop timeline/table view for project template steps and generated project instance milestones.

## Integrations/settings
Dedicated screens for account connection state, sync status, source-of-truth notes, and sync controls.
